# Privacy Policy — GitHub HTML Preview

## 概要

GitHub HTML Preview はユーザーのプライバシーを最重要視して設計されています。
本拡張機能は**一切のユーザーデータを収集・送信しません**。

## データ通信

本拡張機能がネットワーク通信を行う宛先は以下のみです：

- `github.com` — リポジトリ上の HTML ファイルおよび相対アセットの取得
- `raw.githubusercontent.com` および `*.githubusercontent.com` — GitHub がリダイレクトする raw コンテンツ配信先

これらはすべて、ユーザーが明示的に Preview を操作した場合にのみ発生し、
ユーザーの GitHub ログインセッション（Cookie）を使用してアクセスします。

## ローカルデータ

- 描画済み HTML はブラウザのメモリ内（`Map` オブジェクト）にのみ保持されます
- ディスクへの永続化は一切行いません
- タブを閉じるとキャッシュは消去されます
- `chrome.storage` 権限は使用していません

## 外部サービス連携

- アナリティクス：なし
- テレメトリ：なし
- 広告：なし
- 第三者サービスへのデータ送信：なし

## CDN リソース

プレビュー対象の HTML が CDN（KaTeX, Google Fonts, jsDelivr 等）を参照している場合、
サンドボックス内のビューアがそれらを読み込みます。これはプレビュー対象文書自体の挙動であり、
本拡張機能が独自に外部通信を行うものではありません。

## サンドボックス分離

プレビュー実行環境（viewer.html）は Manifest V3 の sandbox ページとして動作し、
opaque origin を持ちます。そのため：

- GitHub の Cookie やセッションにアクセスできません
- 拡張機能の host_permissions を利用できません
- `connect-src https:` が許可されていますが、これは被プレビュー文書自身のスクリプトによる
  通信のみに影響し、拡張機能の認証情報は原理的に漏洩しません
